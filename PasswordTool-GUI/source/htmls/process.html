Detect Patterns

Analyse Raw Data to detect Patterns in them:

* first copy paste raw data into "Raw Input" folder.
* Second select how many thread you want to use.
* Third press "Analyse" button to start processing raw data.
* If you want stop process press "Stop" button.
* You can check output in "Processed Data" folder.
* After That you need to create statistics of this processed data.
* To perform stats generation you will be taken to next tab soon...

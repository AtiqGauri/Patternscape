Here you can detect patterns in a specific password.

Detect Patterns

Analyse raw data to detect patterns:

* copy paste raw data into "Raw Input" folder.
* select how many thread you want to use.
* press "Analyse" button to start processing raw data.
* you can stop process by pressing "Stop" button.
* output can be accessed from "Processed Data" folder.
* once process done, you will be taken to next step(stat generation).

Generate statistics of detected patterns:

* input will be taken from previous step(Analyzed passwords).
* detected patterns(input) can be accessed from "Processed Data" folder.
* select how many thread you want to use.
* press "Statistics" button to start stats generation.
* you can stop process by pressing "Stop" button.

* once process done, you will be taken to next step(import statistics).
